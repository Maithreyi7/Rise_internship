🚀 Host Static Website with S3 & CloudFront

Fast, Global, Serverless Hosting on AWS

🧠 Problem Statement

Websites often face slow speeds and limited uptime due to localized or shared hosting. This leads to poor user experience and SEO issues.

🎯 Objective

Use Amazon S3 to host static files and CloudFront CDN to deliver them globally — ensuring low latency and high availability.

🛠️ Solution Flow

📂 Upload static website (HTML/CSS/JS) to Amazon S3
⚙️ Configure S3 bucket for static website hosting
🔓 Make the bucket content publicly readable via bucket policy
🌐 Create a CloudFront Distribution using the S3 website endpoint as origin
🔁 CloudFront caches and delivers content from edge locations
📡 (Optional) Point your custom domain to CloudFront via Route 53
